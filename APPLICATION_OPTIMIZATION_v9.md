## 🚀 使用指南

### 1. 使用任务系统

```javascript
// 初始化每日任务
dailyTaskSystem.init('daily-task-container');

// 更新任务进度
dailyTaskSystem.updateProgress('quiz_complete', 1); // 完成 1 道题
dailyTaskSystem.updateProgress('study_time', 5); // 学习 5 分钟

// 初始化成就系统
achievementSystem.init('achievement-container');

// 检查成就
achievementSystem.checkAchievement('learning'); // 检查学习成就

// 初始化自习倒计时
studyTimer.init('study-timer-container');

// 控制学习计时
studyTimer.start(); // 开始
studyTimer.pause(); // 暂停
studyTimer.resume(); // 继续
studyTimer.end(); // 结束
```

### 2. 查看数学家详情

```javascript
// 显示数学家详情
showMathematicianDetail(1); // 高斯

// 详情包含：
// - 传记
// - 成就列表
// - 历史事件
// - 名人名言
// - 影响力评价
// - 相关人物
```

### 3. 浏览知识点

```javascript
// 按学科浏览
browseBySubject('algebra'); // 代数
browseBySubject('geometry'); // 几何
browseBySubject('analysis'); // 分析

// 查看知识点详情
showKnowledgeDetail('勾股定理');
```

### 4. 做题练习

```javascript
// 按类型做题
doQuiz('choice'); // 选择题
doQuiz('fill'); // 填空题
doQuiz('calculation'); // 计算题
doQuiz('proof'); // 证明题
doQuiz('application'); // 应用题

// 按难度做题
doQuizByLevel(3); // 金丹期题目
```
