## ⚡ 核心修行模块 | 数学大道，尽在此中

### 📚 开放知识库：九大境界，一览无余
无需解锁，即可遍历数学修仙九大境界，从入门到入圣，所有知识尽数开放：

| 修仙境界 | 对应数学领域 | 核心修行内容 |
|----------|--------------|--------------|
| **练气期** | 算术与逻辑 | 四则运算、数列基础、逻辑推理（万丈高楼平地起，数学大道之基石） |
| **筑基期** | 高等数学 | 微积分基础、极限理论、矩阵运算、概率论（现代科学三大支柱，筑基之关键） |
| **金丹期** | 线性代数与复变函数 | 特征值/特征向量、复平面、柯西积分（多维空间穿梭，实数之外见广阔天地） |
| **元婴期** | 抽象代数 | 群论、环与域、伽罗瓦理论（跳出具体数字，掌握真正的数学家语言） |
| **化神期** | 拓扑与几何 | 点集拓扑、同伦与同调、微分流形（咖啡杯与甜甜圈，形状之本质） |
| **炼虚期** | 数论与代数几何 | 椭圆曲线、模形式、代数簇（整数之纯粹美，现代密码学之钥匙） |
| **合体期** | 朗兰兹纲领 | 自守表示、Galois表示、L-函数（数学大统一理论雏形，连通数论、代数几何与表示论） |
| **大乘期** | 千禧年难题 | P vs NP、纳维-斯托克斯方程、Hodge猜想（站在人类智慧最前沿，解其一即可名垂青史） |
| **渡劫期** | 终极之谜 | 黎曼猜想、连续统假设、大基数公理（数学之边界，素数之奥秘，无穷之层级） |

### 🎮 趣味道场：寓教于乐，轻松增长修为
修行不必苦行僧！我们设有两大趣味道场，在游戏中磨练数学直觉，轻松增长修为：
- **∑ 公式拼图**：将打乱的经典数学公式按正确顺序重组，在拼图中铭记公式之美
- **↬ 数列推演**：观察数列规律，预测下一个数字，在推演中训练数学直觉
