### 5️⃣ 数学知识图谱系统

#### 知识图谱结构

**节点（12 个核心知识点）**：
- 基础层：算术基础、代数基础、几何基础
- 核心层：函数、微积分、线性代数、概率论
- 进阶层：抽象代数、拓扑学、数论、数学分析
- 前沿层：朗兰兹纲领

**边（知识点关联）**：
- 前置关系（prerequisite）：学习某知识前需掌握
- 扩展关系（extension）：某知识的深入发展
- 相关关系（related）：知识点间的横向联系

#### 核心功能

**1. 学习路径规划**
```javascript
// 获取从"算术基础"到"微积分"的学习路径
const path = knowledgeGraph.getLearningPath('arithmetic', 'calculus');
// 返回：[算术基础 → 代数基础 → 函数 → 微积分]
```

**2. 前置知识查询**
```javascript
// 查询"抽象代数"的前置知识
const prereqs = knowledgeGraph.getPrerequisites('abstract_algebra');
// 返回：[代数基础]
```

**3. 后续知识推荐**
```javascript
// 查询掌握"微积分"后可以学习的内容
const advanced = knowledgeGraph.getAdvancedTopics('calculus');
// 返回：[数学分析、微分方程等]
```

**4. 相关知识发现**
```javascript
// 查询与"数论"相关的知识点
const related = knowledgeGraph.getRelatedTopics('number_theory');
// 返回：[{知识点：朗兰兹纲领，关联度：0.8, 类型：前置}]
```
